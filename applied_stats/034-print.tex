\subsection{オブジェクトの画面表示}

\myffr

  \mybfr{手順}
  コンソールに表示させたいオブジェクト名を入力する．
  または，ソースコード画面でオブジェクトを選択してCtrl＋Enterを押す．
  \mybto 

  \myeee
    {コンソール1}
    {
      \cs{v[3]}\\ \relax
      [1] 3
    }
    {コンソール2}
    {
      \cs{m[2, ]}\\ \relax
      [1] 2 4
    }
    {コンソール3}
    {
      \cs{(x <- c(1, 2))}\\ \relax
      [1] 1 2
    }\\[1mm]

  【オブジェクト要素の\R 表記】\\
   ~~ベクトルvの要素$i$： v[i]\\
   ~~行列mの$i$行： m[i, ]，$j$列： m[, j]，要素$(i, j)$： m[i, j]

  \mybfr{演習}
    スカラ，ベクトル，行列の値を表示させてください．
  \mybto
  
\end{frame}
